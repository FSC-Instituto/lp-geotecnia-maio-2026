Como publicar na Hostinger

1. Envie todo o conteudo desta pasta para public_html.
2. Use index.html como pagina inicial em hospedagem estatica.
3. Se quiser uma entrada PHP, use index.php. Ele apenas carrega o index.html.
4. Mantenha a pasta public/assets junto, pois ela contem imagens, videos, favicon e prints.

Arquivos:
- index.html: site principal.
- preview.html: copia HTML separada para previsualizacao.
- index.php: entrada compativel com hospedagem PHP.
- styles.css: estilos.
- app.js: animacoes e interacoes.
- public/assets: midias do site.

Nao precisa subir server.js, package.json ou rodar npm na Hostinger.
